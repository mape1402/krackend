# Krackend

Krackend is a mini framework that allows manage your backend microservices through the tools and abstract class that provides repetitive behaviors like the access data or model validations. Also it helps for administrate the microservices communication using differents communication protocols like HTTP, gRPC, and Message Queues. Everything using clean architecture patterns like SAGAS, Event Sourcing, CQRS, etc.
